# SpectraMap-Engine 🌍🛰️

Dasbor WebGIS interaktif dan responsif (Mobile/Desktop) berbasis **Google Earth Engine (GEE)** untuk analisis geospasial tingkat lanjut.

## 🚀 Fitur Utama
* **Multi-Sensor Support:** Memproses citra Sentinel-2 SR dan Landsat 8/9 SR secara mulus.
* **40+ Kalkulator Indeks:** Menghitung NDVI, EVI, NDBI, NDWI, dan puluhan indeks spektral lainnya secara *on-the-fly*.
* **Smart Cloud Masking:** Menggunakan algoritma canggih *Cloud Score+* (S2) dan *QA_PIXEL* (Landsat) untuk ekstraksi piksel bebas awan.
* **Dashboard Interaktif:** Menampilkan statistik luasan area (Hektar) secara *real-time* lengkap dengan grafik Pie & Bar.
* **Direct Download:** Mendukung ekspor ke tab *Tasks* (Google Drive) dan *Direct Download Link* untuk mengunduh GeoTIFF secara instan.
* **Bilingual UI:** Antarmuka responsif yang mendukung Bahasa Indonesia dan Bahasa Inggris.

1. Salin kode yang ada di file `app.js`.
2. Buka [Google Earth Engine Code Editor](https://code.earthengine.google.com/).
3. Tempel (*paste*) kode tersebut dan klik **Run**.
